Redesign the mobile user interface of the OTTMEGA IPTV React Native / Expo application to make it fully responsive and natural on modern Android mobile devices.

The current mobile layout feels unnatural in both portrait and landscape orientations. The UI must be redesigned so it adapts correctly to different screen sizes and orientations.

The application must support all modern Android devices and full-screen displays.

DEVICE SUPPORT

Ensure the UI works correctly on:

Small Android phones (360px width)

Medium phones (390–420px width)

Large phones (430px+ width)

Foldable devices

Tablets

Landscape phones

The layout must dynamically adapt to screen size.

Use responsive APIs such as:

useWindowDimensions
Dimensions

Define responsive breakpoints:

Small phone: width < 380
Normal phone: 380–420
Large phone: 420–600
Tablet: > 600

Use these breakpoints to adjust layout, grid columns, spacing, and font sizes.
